# IdeaSpark

- Install deps: `npm install`
- Set env vars in `.env` or via Expo: `EXPO_PUBLIC_SUPABASE_URL`, `EXPO_PUBLIC_SUPABASE_ANON_KEY`
- Run: `npm run web` or `npm run android`

Routes
- `/feed` main feed
- `/post` create idea
- `/chat/[id]` DM channel
- `/profile` profile and roles
- `/idea/[id]` idea detail with OG tags
